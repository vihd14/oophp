get()

get(string  $service) : mixed

Finds an entry of the container by its identifier and returns it.

Parameters

string $service

Identifier of the entry to look for.

Throws

No entry was found for this identifier.

Error while retrieving the entry.

Entry.

has()

has(string  $service) : boolean

Returns true if the container can return an entry for the given identifier, otherwise false

has($id) returning true does not mean that get($id) will not throw an exception. It does however mean that get($id) will not throw a NotFoundException.

string $service
